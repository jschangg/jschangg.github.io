What are the benefits of Version Control?

What is "version control", and why should you care? Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. For the examples in this book you will use software source code as the files being version controlled, though in reality you can do this with nearly any type of file on a computer.

How does git help you keep track of changes?

Git lets you keep track of change and lets you go back to any point you would like.

Why should you use GitHub?

There are many reasons why you should use GitHub. The first is that you have a greater control in what VCS you use. The second is that you can have as many private repositories as you would like. Third, you can trust it. When coding, having a trustworthy VC is critical. Lastly, you can use it via app!
